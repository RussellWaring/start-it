Header (regular component .jsx): Top navigation area; logo, links, call-to-actions.
Main (NOT a .jsx file â€” just <main> HTML tag): Primary page content area (between header and footer).
Section (layout .jsx component): Vertical content block with consistent top/bottom spacing.
Container (layout .jsx component): Centers content; controls max-width and side padding.
Stack (layout .jsx component): Vertical layout; evenly spaces children using gap.
Grid (layout .jsx component): Responsive columns/rows layout for arranging blocks.
Card (regular component .jsx): Boxed content unit with padding, border, and radius.
Footer (regular component .jsx): Bottom site area; links, contact, copyright.
