# start-it
A job board web app that helps users browse and explore entry-level opportunities.

## Features
- Browse and filter job postings (placeholder data, for now)
- View job details
- Responsive layout (desktop + mobile)

## Tech Stack
- React
- Vite
- JavaScript / CSS
- GitHub
